## Quick Install

```bash
curl -fsSL https://kyber.chat/install.sh | bash
```

That's it. The installer detects your OS, installs dependencies, walks you through provider setup with model selection, writes your config, and optionally sets up system services.

## Example Workflows

### Debug and Fix a Bug
```
you: "hey kyber, the login endpoint is failing on bad passwords. check it out"
kyber: [reads the code, finds the issue, fixes the validation, adds tests]
```

### Deploy a Project
```
you: "deploy my app to production"
kyber: [runs tests, builds docker image, pushes to registry, updates k8s]
```

### Research and Summarize
```
you: "summarize the latest changes to the OpenAI API docs"
kyber: [fetches docs, extracts key changes, writes summary]
```

### Monitor and Alert
```
you: "check my emails every 30 minutes and tell me about anything urgent"
kyber: [sets up cron job, monitors inboxes, sends alerts to Discord]
```
